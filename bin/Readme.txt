Panduan Instalasi:
1. Setelah download aplikasi (APK) kamus ini, copykan ke folder tempat penyimpanan dimanapun dalam SDCard anda
2. Klik Aplikasi 
3. Pilih Install
4. Tunggu proses instalasi hingga selesai
5. Jika proses instalasi sudah selesai dlakukan, aplikasi siap digunakan.
